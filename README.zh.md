# Next.js 电动车网站模板

一个免费开源的电动摩托车 / 踏板车品牌网站模板，基于 Next.js App Router、React 和 Tailwind CSS 构建。Fork 下来，改成你的，直接上线。

**在线预览：** [tycorunmotorcycle.pages.dev](https://tycorunmotorcycle.pages.dev/)

> ⚠️ 本模板由 AI 辅助开发，本人做了基本校准，但不是专业开发者。能跑，但可能存在一些奇怪的写法、非常规的实现方式，或者让有经验的开发者看了皱眉头的东西。你懂的。

## 技术栈

- Next.js（App Router）
- React
- Tailwind CSS

## 快速开始

```bash
pnpm install
pnpm dev
```

打开 `http://localhost:3000`。

## 构建

```bash
pnpm build
pnpm start
```

## 项目结构

```
app/         # App Router 页面与布局
components/  # UI 组件
data/        # 产品与内容数据
public/      # 静态资源
app/css/     # 全局样式文件
```

## 自定义说明

- 替换 `data/products/*` 里的产品数据
- 替换 `public/images/*` 里的图片
- 在 `app/css/` 里修改品牌色和字体
